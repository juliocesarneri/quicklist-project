# Lista de Compras (QuickList) — Desafio Rocketseat

Este projeto foi desenvolvido como parte de um desafio prático da Rocketseat, com o objetivo de reforçar conceitos fundamentais de HTML, CSS e JavaScript, simulando uma aplicação real de lista de compras.

A aplicação permite adicionar e remover itens dinamicamente, aplicando animações suaves de entrada e saída, além de exibir um toast de feedback sempre que um item é removido da lista.
